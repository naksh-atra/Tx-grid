# User Personas

## Persona 1: AI Agent Orchestrator (Alex)

- Runs 5-10 AI coding agents simultaneously in tmux panes
- Needs to see which agents are working vs. idle
- Wants to quickly jump to a stuck agent or kill runaway processes
- Uses Claude Code, OpenAI Codex, and local LLMs

## Persona 2: Full-Stack Developer (Sam)

- Runs frontend dev server, backend server, tests, and a database
- Wants a quick overview without alt-tabbing through panes
- Occasionally needs to restart a crashed server

## Persona 3: DevOps Engineer (Jordan)

- SSH'd into remote servers with long-running jobs
- Needs to monitor batch jobs, scrapers, and training runs
- Works mostly from the terminal with tmux

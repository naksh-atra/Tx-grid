# Keybindings

## Grid Navigation

| Key | Action |
|-----|--------|
| `j` or `↓` | Move selection down |
| `k` or `↑` | Move selection up |
| `g` or `Home` | Jump to first |
| `G` or `End` | Jump to last |
| `PgUp` | Page up (10 items) |
| `PgDn` | Page down (10 items) |

## Actions

| Key | Action |
|-----|--------|
| `Enter` | Jump to selected pane |
| `x` | Kill selected pane |
| `r` | Restart task in selected pane |
| `/` | Enter filter mode |
| `s` | Cycle sort (runtime→session→state→runtime) |
| `q` or `Esc` | Quit |

## Filter Mode

| Key | Action |
|-----|--------|
| `<type>` | Add character to filter |
| `Backspace` | Remove last character |
| `Enter` | Apply filter, exit filter mode |
| `Esc` | Cancel filter, clear filter text |

## Confirm Dialog

| Key | Action |
|-----|--------|
| `y` | Confirm action |
| `n` or `Esc` | Cancel |
